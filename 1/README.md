# Практическая работа №1: Карточка товара с CSS-препроцессором (SASS)

## Использованные технологии:
- HTML5
- SASS (SCSS)

## Реализованные возможности препроцессора:

### 1. Переменные:
- `$primary-color` - основной цвет
- `$secondary-color` - вторичный цвет
- `$text-color` - цвет текста
- `$border-radius` - радиус скругления
- `$box-shadow` - тень

### 2. Миксины:
- `@mixin button-styles` - стилизация кнопок
- `@mixin card-styles` - стилизация карточек

### 3. Вложенная структура:
- Полная вложенность по методологии БЭМ
- Вложенные псевдоклассы (`&:hover`, `&:active`)
- Вложенные медиа-запросы

## Установка и запуск:
1. Установите зависимости: `npm install`
2. Запустите компиляцию: `npm run sass`
3. Откройте `index.html` в браузере

## Структура проекта:
- `index.html` - основной HTML-файл
- `scss/styles.scss` - исходный SASS-файл
- `css/styles.css` - скомпилированный CSS (создастся автоматически)
