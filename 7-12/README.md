# Управление товарами (RBAC)

Fullstack-приложение на Node.js/Express + React с системой ролей.

## Запуск проекта

### Бэкенд
```bash
cd backend
npm install
# создать файл .env с переменными:
# ACCESS_SECRET=your_access_secret
# REFRESH_SECRET=your_refresh_secret
# PORT=5000
npm run dev
